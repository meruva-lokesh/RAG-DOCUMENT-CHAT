## 🧪 How to Test:

### **1. Upload the PDF**
- Go to: http://localhost:8501
- Click "Browse files" in sidebar
- Upload any PDF

### **2. Wait for Processing**
You'll see:
```
✅ Successfully processed 1 file(s), created XX chunks, extracted Y images 🖼️
```

### **3. Ask Questions**

**Text-based:**
- "What is this document about?"
- "Summarize the main points"

**Image-based:**
- "Show me the diagrams"
- "What charts are included?"
- "Find images about architecture"

**Mixed:**
- "Explain the diagram on page 3"
- "What does the performance chart show?"
